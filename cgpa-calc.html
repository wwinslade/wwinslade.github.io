Cumulative GPA Calculator

  • Back to homepage
  • Last Updated: 9/18/2023

UT has a tool on their website that implements a GPA calculator. The powers that be determined that 20 course fields would be sufficient for figuring out your GPA, with no option to add more input fields. That sucks, and it requires an internet connection, so I wrote my own. Beware, some institutions have different GPA scales than UT, so results may vary if you are not a UT student. I've included UT's table here:

Grade Value
A 4.0
A- 3.7
B+ 3.3
B 3.0
B- 2.7
C+ 2.3
C 2.0
C- 1.7
D+ 1.3
D 1.0
D- 0.7
F 0.0

Usage

If you want to use this program you can download the source code here. Simply compile the source, and call on the command line with: 
./a.out grades_file.txt
The format of the text file you need to give it isn't strict. So long as for each course you are inputting, you list the hours, followed by a space, then grade you received in the class, it'll work as intended. Grades are not case sensitive. The program prints your total hours, total grade points, and cumulative GPA to standard output in your shell environment.

For example, either of the below would work fine:
example1.txt example2.txt
2 a
3 b
4 a
2 c+
2 A 3 B 4 A 2 C+
