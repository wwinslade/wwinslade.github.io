Boyer-Moore Majority Vote

The Boyer-Moore majority vote algorithm is a streaming algorithm that is useful for finding the majority element in a sqeuence of elements. It is notable because it accomplishes this in linear time and a constant amount of memory.

The algorithm is implemented by keeping track of an element m, and a counter variable c. To setup the algorithm, initialize c as zero. The algorithm then iterates through the list and examines each element. If c is zero, then we set m to be the ith element. Otherwise, if the ith element matches m, then we increment c by one, and if it doesn't match m, then we decrement c by one.

This process will return a candidate that could be the majority element, but it is not guaranteed. To verify the candidate, we must perform a second pass of the elements, and count frequencies to verify that the element is indeed the majority.

See the below C++ code for a basic implementation, which finds the majority element of a vector. 
#include <iostream>
#include <vector>

using namespace std;

int Boyer_Moore_Majority_Vote(const vector <int>& nums) {
        
    int i, m, c;
    
    c = 0;
    for (i = 0; i < nums.size(); i++) {
        if (c == 0) {
            m = nums[i];
            c = 1;
        } else {
            if (nums[i] == m) {
                c++;
            } else {
                c--;
            }
        }
    }
    return m;
}
